A* Search Concept

What is the 8-Puzzle?

The 8-puzzle is a sliding puzzle consisting of a 3x3 grid of numbered tiles in random order with one tile missing. The goal is to place the tiles in order (1 to 8, with the blank space at the end) by sliding tiles into the empty space.

What is A* Search?

A* is a path-finding algorithm used to find the shortest path from a start configuration to the goal state. It calculates a cost value f(n) for neighboring states and chooses the path that minimizes:

f(n) = g(n) + h(n)
  • g(n): The path cost (number of moves made from the start state to the current state).
  • h(n): The heuristic cost (estimated cost to reach the goal state). We use the Manhattan Distance.

What is Manhattan Distance?

For each tile, it counts the absolute horizontal + vertical distance to its target goal position:
distance = |x1 - x2| + |y1 - y2|.
Adding these distances for all 8 numbered tiles gives h(n). A heuristic cost of 0 means the puzzle is solved!
